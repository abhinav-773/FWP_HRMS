# HRGPT

AI-powered HRMS SaaS platform.

## Architecture

* **Frontend**: React, Tailwind CSS, Framer Motion
* **Backend**: Node.js, Express, Socket.io
* **AI Service**: Python, FastAPI
* **Databases**: PostgreSQL, MongoDB

## Prerequisites

* Node.js (v18+)
* Python 3.10+
* Docker & Docker Compose
