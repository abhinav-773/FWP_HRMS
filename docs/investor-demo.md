# HRGPT Investor Demo Script

## The Goal
To demonstrate HRGPT's core value proposition: An autonomous HR platform that drastically reduces manual HR overhead (time-to-hire, support tickets, payroll processing) through AI.

## Flow

### 1. Executive Dashboard (0:00 - 2:00)
- **Role:** Login as Super Admin (CEO Persona)
- **Navigate to:** Executive Overview
- **Talking Points:**
  - "HRGPT provides top-down visibility into how AI is accelerating the business."
  - Highlight the **AI Cost Savings** metric and **Time-to-Hire** reduction.
  - Show the **AI Automation Impact** chart, demonstrating how manual hours are decreasing while AI-assisted output rises.

### 2. Autonomous ATS (2:00 - 5:00)
- **Navigate to:** ATS Pipeline (`/hr/pipeline`)
- **Talking Points:**
  - "This is where the magic happens. HRGPT autonomously screens applications."
  - Point out the **AI Score** badges on candidate cards.
  - Show the **AI Pipeline Visualizer** to explain the backend process (Resume Parsing -> Embeddings -> Llama 3 Evaluation).
  - Drag a candidate to the "Interview" stage and explain that HRGPT can auto-trigger asynchronous AI voice interviews.

### 3. AI Copilot (5:00 - 7:00)
- **Action:** Open the global AI Copilot widget (bottom right).
- **Talking Points:**
  - "Instead of digging through menus, executives and managers can just ask the Copilot."
  - **Prompt:** *"Show me the top 3 candidates for the Frontend Developer role."*
  - **Prompt:** *"Who is on leave today across the engineering team?"*
  - Explain how the Copilot understands the context of the user and the page they are on.

### 4. Real-time Collaboration (7:00 - 8:00)
- **Action:** Demonstrate a live notification arriving (e.g., "Candidate completed AI interview").
- **Talking Points:**
  - "Everything in HRGPT is real-time. Notifications, ATS movements, and team chat ensure remote teams stay perfectly in sync."

### 5. Closing
- Reiterate that HRGPT is fully deployable in enterprise environments (can be self-hosted with local LLM inferencing for ultimate data privacy).
